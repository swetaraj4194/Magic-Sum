Magic Sum

What is Magic Sum?

This is a very interesting game. In this game you can see sum result before compeleting the sum.

Rules

Please go through step by step:

  1. Enter any 5 digit number between 2 to 9.

  2. Now you can check your sum result under pre-result section.

  3. Again Enter any 5 digit numbers between 2 to 9.

  4. Now system will enter 5 digit number.

  5. Once again enter any 5 digit number between 2 to 9.

  6. Again system will enter 5 digit number.

  7. Now you can check final sum result and the magic is Pre-Result sum and Final Result are same.

User Input





Enter any 5 digit number between 2 to 9 and click on submit.
